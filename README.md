# Empório das Ferragens - Site Estático

Este diretório contém os arquivos prontos para publicação no **GitHub Pages**.

## Como publicar:

1. Crie um novo repositório no GitHub.
2. Suba o conteúdo deste diretório (`dist`) para o repositório.
3. Vá em **Settings** > **Pages**.
4. Em **Build and deployment**, selecione a branch `main` (ou a que você subiu) e a pasta `/(root)`.
5. Clique em **Save**.

## Estrutura:
- `index.html`: Página principal.
- `404.html`: Cópia do index para suportar rotas do React (SPA) no GitHub Pages.
- `assets/`: Arquivos CSS e JS compilados.
- `images/`: Imagens do site.
- `robots.txt` e `sitemap.xml`: Arquivos para SEO.
